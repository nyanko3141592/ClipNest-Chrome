ClipNest — クリップボード履歴 & スニペット

ブラウザ内で動く高速・軽量なクリップボードマネージャー。コピーした内容の履歴を自動保存し、よく使うテキストをスニペットとして管理。右クリックメニューからすぐにペーストできます。

━━━ 機能 ━━━

🖱️ 右クリックメニュー（メインUI）
• テキストフィールドを右クリック → 履歴＆スニペットにすぐアクセス
• フォルダはサブメニューとしてそのまま展開
• ピン留めスニペット（★）は常に先頭に表示
• ページ上のテキストを選択 → 右クリック → スニペットとして保存
• 履歴のクリアもメニューから直接可能

📋 クリップボード履歴
• コピーしたテキストを自動でキャプチャ
• 右クリックメニューに最大15件表示
• 履歴サイズは 10〜100 件で設定可能

✂️ スニペット管理
• テキストを名前付きスニペットとして保存
• ネストしたフォルダで整理
• 重要なスニペットをピン留め
• 動的プレースホルダー: {{date}}, {{time}}, {{datetime}}

⌨️ キーボードショートカット
• Alt+V（カスタマイズ可能）でカーソル横にインラインパネルを表示
• 矢印キー・Enter・Tab でマウスなし操作
• / で検索、e で編集、s で履歴をスニペット保存

🖱️ ツールバーポップアップ
• ClipNest アイコンをクリックで履歴＆スニペットを閲覧・管理
• chrome:// などの制限ページでも動作

🔒 プライバシー
• すべてのデータはブラウザ内にローカル保存（chrome.storage.local）
• アカウント不要・サーバー通信なし・トラッキングなし
• 外部依存ゼロ — 純粋な vanilla JS

━━━ 使い方 ━━━

1. いつも通りテキストをコピー — ClipNest が自動でキャプチャ
2. テキストフィールドで右クリック → ClipNest → 履歴やスニペットを選択
3. テキストがフィールドに直接ペーストされます
4. ページ上のテキストを選択 → 右クリック → スニペットとして保存

━━━ キーボードショートカット（インラインパネル）━━━

↑ ↓       アイテム移動
Enter     ペースト
1–9       クイック選択
Tab       Snippets ↔ History 切替
→ ←       フォルダ展開 / 折りたたみ
/         検索トグル
e         選択アイテム編集
s         履歴をスニペットに保存
Esc       閉じる

━━━ 権限について ━━━

• storage — 履歴とスニペットのローカル保存
• activeTab / scripting — インラインパネルの表示とテキストのペースト
• contextMenus — 右クリックメニューへの ClipNest の追加
